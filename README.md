### Installation

1. Clone the repo

```sh
   git clone https://github.com/ViuCoding/universal-plastic-vincenzo-cristiano.git
```

2.  Install NPM packages

```sh
npm install
```

This will install the necessary dependencies to run the program.

3. Launch a local server:

```sh
  npm run dev
```
